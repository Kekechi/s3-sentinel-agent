The North Star: S3-Sentinel-Graph is a security-orchestration project that implements a Functional Gatekeeper pattern. Unlike traditional LLM wrappers, this system decouples agentic reasoning from execution authority. By utilizing LangGraph's GraphAPI for the core state machine and restricting LangChain to a communication interface, the project ensures that security boundaries (Admin vs. Guest) are enforced by hard-coded Python logic that is immune to LLM prompt injection or "hallucinated" permissions.
